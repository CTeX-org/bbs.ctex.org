# CTeX 临时论坛

由于 [原 CTeX 论坛](http://bbs.ctex.org/) 因故无限期关闭，我们在此建立临时论坛，通过 GitHub Issues 进行提问、回答与讨论。

在此处提问、讨论请遵循 [GitHub 社区规范](https://help.github.com/en/articles/github-community-guidelines)。
